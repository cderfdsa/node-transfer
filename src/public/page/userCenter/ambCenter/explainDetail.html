绑定用户的方式、规则与好处？

总业绩 = 本人业绩 + 邀请大使业绩 + 关联业绩

总收益 = 本人收益 + 邀请大使收益 + 关联收益 + 高级任务收益 + 代言费 + 邀请好友收益

校园大使权益有效期2年，升级根据你的业绩而定。


注：超越规则：下属大使级别高于大使本人后，保护期为45天，若45天后级别依旧低于下级大使，则绑定关系取消（同级不受影响）； 已邀请成功的下属大使，除非超越，否则绑定关系将一直持续，无特殊原因不解绑。

升级后可享受更多的特权，不仅购物可以享受额外VIP折扣， 提成收益的比例更高，还可获得更多现金奖励、美妆独家福利等。 具体见等级说明.


每成功邀请一位用户成为“你的大使”，可以获得一次性邀请奖励50-100元（被邀请人选择350元现金抵用券，返收益50元；选择美妆礼包，返收益100元），并且今后该大使的所有业绩你都可以获得收益提成。邀请大使无上限数量限制，邀请越多，收益越高。


邀请好友成功可获得100颜值奖励，并且TA在美妆完成首单购买后，你可获得10元现金奖励。


规则：新版规则里，不同等级的校园大使可绑定的用户人数不一样。等级越高，可绑定的用户名额越多。


方式：用户在【个人中心 → 个人信息】页面输入你的手机号，即算与你绑定成功；


好处：绑定用户成功后，今后该用户购买的所有订单你都享有高额直接业绩提成。
