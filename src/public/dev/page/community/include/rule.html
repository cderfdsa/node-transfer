美妆商城致力于为高校女生提供一个高质量的分享美的平台。为了维护良好的社区氛围，美妆依照本规定中的条款对注册用户提供的内容和行为进行管理和服务。

违规内容和行为有哪些？

美妆是一个以女大学生坚持变美为核心的社交分享平台，鼓励用户在此交流讨论与美相关话题，如下几项内容是社区不允许的。

  1. 1、违反法律法规：

    发布违反法律法规，危害国家及社会安全的信息，主要表现为：
    • 反对宪法所确定的基本原则
    • 危害国家安全，泄露国家秘密，颠覆国家政权，破坏国家统一
    • 损害国家荣誉和利益
    • 煽动民族仇恨、民族歧视，破坏民族团结
    • 破坏国家宗教政策，宣扬邪教和封建迷信
    • 散布谣言，扰乱社会秩序，破坏社会稳定
    • 宣扬淫秽、色情、赌博、暴力、凶杀、恐怖或者教唆犯罪
    • 煽动非法活动、结社、游行、示威、聚众扰乱社会秩序
    • 诽谤他人，泄露他人隐私，侵害他人合法权益
    • 含有法律、行政法规禁止的其他内容的信息

  2. 2、不友善行为：

    不尊重美妆用户及其所贡献内容的行为，主要表现为：
    • 轻蔑：贬低、轻视他人及其劳动成果
    • 诽谤：捏造、散布虚假事实，损害他人名誉
    • 嘲讽：以比喻、夸张、侮辱性的手法对他人或其行为
    • 进行揭露或描述，以此来激怒他人
    • 挑衅：以不友好的方式激怒他人，意图使对方对自己的言论作出回应，蓄意制造事端
    • 羞辱：贬低他人的能力、行为、生理或身份特征，让对方难堪
    • 谩骂：以不文明的语言对他人进行负面评价
    • 恶意举报：因为个人恩怨或者主观情绪举报他人
    • 社会歧视：对他人的民族、种族、宗教、性取向、性别、年龄、地域、生理特征等身份或者内容进行攻击
    • 威胁：许诺以不良的后果来迫使他人服从自己的意志

  3. 3、发布垃圾内容：

    发布影响用户体验、与社区氛围不符的内容，主要表现为：
    • 发布重复内容：包括同一内容实名、匿名或是小号各发一遍；以评论、回复评论等方式对他人反复发送色情、段子鸡汤、广告等内容
    • 灌水：发布与美无关的鸡汤、段子，一连串没意义的文字、符号、字母、表情等内容
    • 含有淫秽色情、暴力、血腥、仇恨的内容
    • 发布含有钓鱼网站、木马、病毒网站链接及相关内容
    • 发布含有潜在危险、窃取用户隐私的相关内容
    • 机器抓取发送的无意义内容、重复内容
    • 发布和社区基调不符的图片，如恶意搞怪图片/影响美观图片等
    • 以下几种情况也视为无效信息，会被管理员删除,代言笔记发鸡汤段子、招聘、求职、婚恋内容的，以及其他和美妆心得/分享/教程无关的内容

  4. 4、发布广告推销内容:

    • 包括但不限于在社区内以文字、视频、图片、二维码、站外链接的形式从事发帖、发评论、转发活动的。
    • 通过机器或软件等非人力手段向他人页面大量发送广告信息
    • 通过错误的图文搭配进行广告宣传

  5. 5、账号违规操作:

    • 通过注册多个帐号来谋求不正当的利益。包括但不限于升级等
    • 恶意伪造、上传通讯录来抬高影响力，制造虚假人脉关系
    • 买卖账号、将账号借用给他人

个人账号只允许本人使用。买卖、转借个人账号，一经发现，我们有权停用此账号，由此造成的一切损失由账号申请者承担。通过多个账号谋求不正当利益的，一经发现，禁用所有账号。

美妆通过主动发现和接受用户举报两种方式发现违规行为。用户可以使用美妆页面上的“举报”功能对涉嫌违规的内容和户进行举报。举报后由美妆管理员人为审核并处理。因在其他平台发生的纠纷、矛盾以及利益受损而产生的举报行为，美妆平台不予受理。

违规内容如何处理？

  1. 1、删除与警告:

    • 多数情况下管理员会对首次发布的不恰当内容发出警告，并给予用户改正自己不当行为的机会，但不表示美妆团队在采取任何措施前必须首先发出警告。对任何非法、不良和引起他人不适的内容或信息，美妆有权先行删除.

  2. 2、封禁和注销：

    发布上述违规内容，除了删除和警告外，账号还可能被封禁乃至注销。
    • 封禁的权限包括：禁全站、禁PO图、禁代言笔记、禁评论、禁活动，如果某用户同时触发任意两项权限，将被全站封禁。
    • 封禁期限包括：一天、一周和永久，视违规次数或者违规程度而定。

  3. 3、申诉：

    如果用户对美妆基于本规范做出的处理有异议，可通过邮件ilovemw@meiwan.me 申诉。申诉时写明姓名、注册手机号以及申诉事项。
